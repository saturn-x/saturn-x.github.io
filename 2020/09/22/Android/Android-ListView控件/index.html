иҝҗиЎҢз»“жһң

йҰ–е…Ҳе®ҡд№үдәҶдёҖдёӘеӯ—з¬ҰдёІж•°з»„еӯҳж•°жҚ®пјҢж•°з»„дёӯзҡ„ж•°жҚ®жҳҜж— жі•зӣҙжҺҘдј йҖ’з»ҷListViewзҡ„пјҢд»Қз„¶йңҖиҰҒеҖҹеҠ©йҖӮй…ҚеҷЁ

е®һзҺ°иҝҮзЁӢпјҡж–°е»әйҖӮй…ҚеҷЁ->ж·»еҠ ж•°жҚ®жәҗеҲ°йҖӮй…ҚеҷЁ->и§ҶеӣҫеҠ иҪҪйҖӮй…ҚеҷЁ

AndroidдёӯжҸҗдҫӣдәҶеҫҲеӨҡйҖӮй…ҚеҷЁзҡ„е®һзҺ°зұ»пјҢиҝҷж¬ЎдҪҝз”Ёзҡ„жҳҜArrayAdapterпјҢеҸҜд»ҘйҖҡиҝҮжіӣеһӢжқҘжҢҮе®ҡиҰҒйҖӮй…Қзҡ„ж•°жҚ®зұ»еһӢпјҢз„¶еҗҺеҶҚжһ„йҖ еҮҪж•°дёӯжҠҠиҰҒйҖӮй…Қзҡ„ж•°жҚ®дј е…ҘгҖӮArrayAdapterжңүеӨҡдёӘжһ„йҖ еҮҪж•°йҮҚиҪҪпјҢз”ұдёҺжҲ‘д»¬жҸҗдҫӣзҡ„ж•°жҚ®йғҪдёәеӯ—з¬ҰдёІпјҢжүҖжңүжіӣеһӢжҢҮе®ҡдёәStringпјҢз„¶еҗҺеҶҚArrayAdapterзҡ„жһ„йҖ еҮҪж•°дёӯдҫқж¬Ўдј е…Ҙ еҪ“еүҚзҡ„дёҠдёӢж–ҮгҖҒListViewзҡ„еӯҗйЎ№еёғеұҖзҡ„id д»ҘеҸҠиҰҒйҖӮй…Қзҡ„ж•°жҚ®

иҝҷйҮҢдҪҝз”ЁдәҶandroid.R.layout.simple_list_item_1дҪңдёәListViewеӯҗйЎ№еёғеұҖзҡ„idпјҢиҝҷжҳҜдёҖдёӘAndroidеҶ…зҪ®зҡ„еёғеұҖж–Үд»¶пјҢйҮҢйқўеҸӘжңүдёҖдёӘTexViewпјҢеҸҜз”ЁдәҺжҳҫзӨәдёҖж®өж–Үжң¬пјҢиҝҷж ·йҖӮй…ҚеҷЁзҡ„еҜ№иұЎе°ұжһ„е»әеҘҪдәҶ

жңҖеҗҺпјҢйҖҡиҝҮи°ғз”ЁListViewзҡ„setAdapter( )ж–№жі•пјҢе°Ҷжһ„е»әеҘҪзҡ„йҖӮй…ҚеҷЁеҜ№иұЎдј йҖ’иҝӣеҺ»пјҢиҝҷж ·ListViewе’Ңж•°жҚ®д№Ӣй—ҙзҡ„е…іиҒ”е°ұе»әз«Ӣе®ҢжҲҗ

2.е®ҡеҲ¶ListViewз•Ңйқў

е®һзҺ°еҺҹзҗҶпјҡйҖҡиҝҮеҲӣе»әдёҖдёӘиҮӘе®ҡд№үзұ»пјҢзұ»дјјдәҺStringзұ»еһӢпјҢз„¶еҗҺеҜ№иҝҷдёӘиҮӘе®ҡд№үзұ»иҝӣиЎҢдёҖдёӘеёғеұҖж–Үд»¶зҡ„еҲӣе»әпјҢиҝҷдёӘеёғеұҖж–Үд»¶дё»иҰҒз”ЁеңЁе»әз«ӢдёҖдёӘйҖӮй…ҚеҷЁзҡ„жҺ’еҲ—ж–№ејҸпјҢз„¶еҗҺе»әз«ӢдёҖдёӘеҹәдәҺArrayAdapterзҡ„дёҖдёӘжҙҫз”ҹзұ»пјҢйҮҚеҶҷйҮҢйқўзҡ„жһ„йҖ еҮҪж•°д»ҘеҸҠgetView( )еҮҪж•°

е®ҡд№үдёҖдёӘе®һдҪ“зұ»пјҢдҪңдёәListViewйҖӮй…ҚеҷЁзҡ„йҖӮй…Қзұ»еһӢпјҢж–°е»әhero hero.javaд»Јз ҒеҰӮдёӢпјҡ

йңҖиҰҒдёәListViewзҡ„жҜҸдёҖдёӘеӯҗйЎ№жҢҮе®ҡдёҖдёӘеёғеұҖж–Үд»¶пјҢж–°е»әhero_item.xml д»Јз ҒеҰӮдёӢпјҡ

еңЁиҝҷдёӘеёғеұҖдёӯдё»иҰҒе®ҡд№үзҡ„жҳҜдёҖдёӘImageViewз”ЁдәҺеҺ»жҳҫзӨәиӢұйӣ„зҡ„еӣҫзүҮпјҢиҖҢдё”е®ҡд№үдәҶдёҖдёӘTextViewз”ЁдәҺжҳҫзӨәиӢұйӣ„зҡ„еҗҚз§°пјҢе№¶дё”и®©TextViewеңЁеһӮзӣҙж–№еҗ‘дёҠеұ…дёӯжҳҫзӨә

жҺҘдёӢжқҘзұ»дјјдәҺдёҠйқўзҡ„жіӣеһӢйҖӮй…ҚеҷЁпјҢйңҖиҰҒиҮӘе®ҡд№үдёҖдёӘйҖӮй…ҚеҷЁпјҢиғҪе®№зәіheroиҝҷдёӘзұ»зҡ„пјҢж–°е»әHeroAdapter.java

public HeroAdapter(Context context, int textViewResouceId, List<hero> objects)йҮҚеҶҷдәҶзҲ¶зұ»зҡ„дёҖз»„жһ„йҖ еҮҪж•°пјҢе°ҶдёҠдёӢж–ҮпјҢListViewеӯҗйЎ№зҡ„еёғеұҖзҡ„idе’Ңж•°жҚ®гҖӮ

еҸҰеӨ–йҮҚеҶҷдәҶgetView( )ж–№жі•пјҢйҰ–е…ҲйҖҡиҝҮgetItemж–№жі•еҫ—еҲ°еҪ“еүҚзҡ„heroе®һдҫӢпјҢз„¶еҗҺдҪҝз”ЁLayoutInflater.from(getContext()).inflate(resouceId,patrent,false)еҠ иҪҪдј е…ҘеёғеұҖ

и°ғз”ЁViewзҡ„findViewId( )иҺ·еҸ–еҲ°Image

Viewе’ҢTextViewе®һдҫӢпјҢ з”ЁдёӢеҲ—иҜӯеҸҘ

жҳҫзӨәеӣҫзүҮе’Ңж–Үеӯ—пјҢжңҖеҗҺе°ҶеёғеұҖиҝ”еӣһпјҢиҝҷж ·иҮӘе®ҡд№үзҡ„йҖӮй…ҚеҷЁе°ұе®ҢжҲҗ

жңҖеҗҺеҶҚMainActivityдёӯзҡ„д»Јз Ғпјҡ

йҰ–е…Ҳз”Ёintihero( )еҺ»еҲқе§ӢжүҖжңүзҡ„иӢұйӣ„,е°Ҷheroзҡ„жһ„йҖ еҮҪж•°е°Ҷж°ҙжһңе’ҢеӣҫзүҮidдј е…ҘпјҢз„¶еҗҺжҠҠеҲӣе»әеҘҪзҡ„еҜ№иұЎж·»еҠ еҲ°ж°ҙжһңеҲ—иЎЁдёӯпјҢжҺҘзқҖonCreate( )ж–№жі•еҲӣе»әдәҶHeroAdapterеҜ№иұЎпјҢжңҖеҗҺе°ҶadapterдҪңдёәйҖӮй…ҚеҷЁдј з»ҷListView иҝҷж ·е°ұе®ҢжҲҗ

3.жҸҗеҚҮListViewиҝҗиЎҢйҖҹзҺҮ

е…¶дёӯеңЁgetView( )ж–№жі•иҝӣиЎҢдәҶеҲӨж–ӯпјҢеҰӮжһңconverViewдёәnullпјҢеҲҷдҪҝз”ЁLayoutInflaterеҺ»еҠ иҪҪеёғеұҖпјҢеҰӮжһңдёҚдёәnullеҲҷзӣҙжҺҘеҜ№convertViewиҝӣиЎҢйҮҚз”ЁпјҢиҝҷж ·жҸҗй«ҳдәҶListViewзҡ„иҝҗиЎҢж•ҲзҺҮгҖӮжңҖйҮҚиҰҒдҪҝз”ЁдәҶview=converView

4.ListViewзӮ№еҮ»дәӢд»¶

дҪҝз”ЁдәҶlistView.setOnItemClickListener( )и®ҫзҪ®дәҶдёҖдёӘзӣ‘еҗ¬еҷЁпјҢеҪ“з”ЁжҲ·зӮ№еҮ»дәҶListViewд»»дҪ•дёҖдёӘеӯҗйЎ№ж—¶пјҢе°ұдјҡеӣһи°ғonItemClick( )ж–№жі•пјҢеңЁиҝҷдёӘж–№жі•дёӯеҸҜд»ҘйҖҡиҝҮpositionеҸӮж•°еҲӨж–ӯеҮәз”ЁжҲ·зӮ№еҮ»зҡ„жҳҜе“ӘдёҖдёӘеӯҗйЎ№пјҢз„¶еҗҺиҺ·еҫ—зӣёеә”зҡ„з»“жһңгҖӮ

5.RecycleViewзҡ„еҹәжң¬з”Ёжі•

е’ҢзҷҫеҲҶжҜ”еёғеұҖзұ»дјјпјҢRecycleViewд№ҹеұһдәҺж–°еўһзҡ„жҺ§д»¶пјҢдёәдәҶи®©RecyclerViewеңЁжүҖжңүAndroidзүҲжң¬дёҠйғҪиғҪдҪҝз”ЁпјҢе°ҶRecycleVuewе®ҡд№үеңЁдәҶsupportеә“дёӯпјҢеӣ жӯӨпјҢжғіиҰҒRecycleViewиҝҷдёӘжҺ§д»¶пјҢйҰ–е…ҲйңҖиҰҒеңЁйЎ№зӣ®зҡ„build.gradleж·»еҠ зӣёеә”зҡ„дҫқиө–еә“

жү“ејҖapp/build.gradleж–Үд»¶пјҢеңЁdependenciesй—ӯеҢ…дёӯж·»еҠ е…ҘдёӢеҶ…е®№:

жҺҘдёӢжқҘйңҖиҰҒдёәRecyclerViewеҮҶеӨҮдёҖдёӘйҖӮй…ҚеҷЁпјҢж–°е»әheroAdapterзұ»пјҢи®©иҝҷдёӘйҖӮй…ҚеҷЁз»§жүҝиҮӘRecycleView.AdapterпјҢе№¶жіӣеһӢжҢҮе®ҡдёәHeroAdapter.ViewHolderпјҢе…¶дёӯпјҢHeroAdapter.ViewHolderжҳҜеңЁheroAdapterдёӯе®ҡд№үзҡ„дёҖдёӘеҶ…йғЁзұ»пјҡ

йҰ–е…Ҳе®ҡд№үдәҶдёҖдёӘеҶ…йғЁзұ»ViewHolderпјҢViewHolderиҰҒз»§жүҝиҮӘRecycleView.ViewHolderгҖӮз„¶еҗҺViewHolderзҡ„жһ„йҖ еҮҪж•°дёӯиҰҒдј е…ҘдёҖдёӘViewеҸӮж•°пјҢиҝҷдёӘеҸӮж•°йҖҡеёёжҳҜRecycleViewеӯҗйЎ№зҡ„жңҖеӨ–еұӮеёғеұҖпјҢжңҖеҗҺеҸҜд»ҘйҖҡиҝҮfindViewById( )ж–№жі•жқҘиҺ·еҸ–еёғеұҖдёӯзҡ„ImageViewе’ҢTextView( )зҡ„е®һдҫӢ

з”ұдәҺHeroAdapterжҳҜз»§жүҝиҮӘRecyclerView.Adapterзҡ„пјҢйӮЈе°ұеҝ…йЎ»йҮҚеҶҷе…¶дёӯзҡ„дёүдёӘж–№жі•

public ViewHolder onCreateViewHolder( )з”ЁдәҺеҲӣе»әдёҖдёӘViewHolderе®һдҫӢпјҢе№¶жҠҠеҠ иҪҪеҮәжқҘзҡ„еёғеұҖдј е…ҘеҲ°жһ„йҖ еҮҪж•°еҪ“дёӯпјҢжңҖеҗҺе°Ҷзҡ„е®һдҫӢиҝ”еӣһгҖӮ

public void onBindViewHolder( )ж–№жі•йҖӮз”ЁдәҺеҜ№RecyclerViewеӯҗйЎ№зҡ„ж•°жҚ®иҝӣиЎҢиөӢеҖјпјҢдјҡеңЁжҜҸдёӘеӯҗйЎ№иў«ж»ҡеҠЁеҲ°еұҸе№•еҶ…зҡ„ж—¶еҖҷжү§иЎҢпјҢиҝҷйҮҢжҲ‘д»¬йҖҡиҝҮpositionеҸӮж•°еҫ—еҲ°еҪ“еүҚйЎ№зҡ„heroе®һдҫӢпјҢз„¶еҗҺе°Ҷж•°жҚ®и®ҫзҪ®еҲ°ViewHolderзҡ„ImageViewе’ҢTextViewеҪ“дёӯгҖӮ

public int getItemCount( )з”ЁдәҺе‘ҠиҜүRecyclerViewдёҖе…ұжңүеӨҡе°‘еӯҗйЎ№пјҢзӣҙжҺҘж”ҫеӣһж•°жҚ®жәҗзҡ„й•ҝеәҰ
